# Jenna Eve Portfolio

A modern, minimalist portfolio website for Jenna Eve, a Visual Designer & Creative Strategist.
This project is built with React and Vite, featuring a clean design inspired by Maxine Ancheta.

## Features

*   **Modern Layout:** Asymmetric design with a fixed header and floating navigation dock.
*   **Interactive Elements:** Custom cursor, smooth scroll animations, and hover effects.
*   **Responsive Design:** Fully responsive layout that adapts to mobile and desktop screens.
*   **Typography:** Carefully curated typography using Inter (Sans-Serif) and Noto Serif (Serif).

## Tech Stack

*   [React](https://reactjs.org/)
*   [Vite](https://vitejs.dev/)
*   [Framer Motion](https://www.framer.com/motion/) for animations
*   [React Router](https://reactrouter.com/) for navigation
*   CSS Modules / Vanilla CSS for styling

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/nmang004/jennaportfolio.git
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:5173](http://localhost:5173) (or the port shown in your terminal) to view it in the browser.

## Project Structure

*   `src/components`: Reusable UI components (Header, ProjectCard, etc.)
*   `src/pages`: Page components (Home, About)
*   `src/data`: Static data files (projects.js)
*   `src/assets`: Images and other static assets
